CSS селектор

 пример описание

.class

.myclass выбирает все элементы с класом myclass

id

 myid выберает элемент с id myid

*

 * выделяет все элементы

тег

 div выберает все теги div

элемент элемент

 div h2 выбдирает все элементы <divr< и все элементы <h2<

атрибут

 href выбирает все элементы у которых эсть href

атрибут="значение"

 href~="https://yandex.ru" выберает все элементы у которых атрибут href точно совподает с указаным значением

атрибут~="значение"

 href~="yandex.ru" выберает все элементы c атрибут href в значении которого (в любом месте) встречается подстрока "yandex.ru" в виде отдельного слова

элемент>элемент

 div+p выбирает все элементы <divr< которые расположены сразу перед элементом <p<

элемент~элемент

 p~div выбирает все элементы <divr< которые расположены сразу после элементом <p<

элемент

 td:nth-child(1) выбирает все элементы <td< которые первые в своей группе
